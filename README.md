## what is is
Basically all that this is is a manager for [sheepit](https://sheepit-renderfarm.com), all it really does is add a gui to the .jar with some extra stuffs, like multiple gpus on one interface, auto restarts if the client crashes, discord integration, etc. its not amazing, but it "works" '

## usage
